# 🥗 DayBite — בוט התזונה האישי שלך בטלגרם

DayBite הוא בוט טלגרם חכם לניהול תזונה יומית. הבוט מנתח מה אכלת, מחשב ערכים תזונתיים, ומסייע לתכנן ארוחות מותאמות אישית — הכל בעברית מלאה.

---

## דרישות מוקדמות

- **Node.js גרסה 18 ומעלה** — [הורדה](https://nodejs.org/)
- **חשבון טלגרם** ויצירת בוט דרך [@BotFather](https://t.me/BotFather)
- **מפתח API של Anthropic** — [הרשמה](https://console.anthropic.com/)

---

## יצירת הבוט ב-BotFather

1. פתח טלגרם וחפש `@BotFather`
2. שלח `/newbot`
3. בחר שם לבוט (למשל: `DayBite`)
4. בחר שם משתמש (חייב להסתיים ב-`bot`, למשל: `daybite_mybot`)
5. קבל את ה-**Token** ושמור אותו

---

## שלבי התקנה

### 1. שכפל את הפרויקט

```bash
git clone <repo-url>
cd daybite-bot
```

### 2. התקן תלויות

```bash
npm install
```

### 3. הגדר משתני סביבה

העתק את קובץ הדוגמה וערוך אותו:

```bash
cp .env.example .env
```

פתח את קובץ `.env` וערוך:

```env
TELEGRAM_BOT_TOKEN=הכנס_כאן_את_הטוקן_מ_BotFather
ANTHROPIC_API_KEY=הכנס_כאן_את_מפתח_Anthropic
```

## הרצה מקומית

```bash
npx ts-node src/index.ts
```

או דרך ה-script המוגדר:

```bash
npm start
```

תראה בקונסול:
```
🤖 DayBite Bot מתחיל...
✅ פקודות הבוט עודכנו ב-BotFather
✅ DayBite Bot פועל! שלח /start בטלגרם כדי להתחיל.
```

## הוספת משתמש חדש

פשוט שלח הודעה לבוט בטלגרם!

1. פתח את הבוט בטלגרם (חפש לפי שם המשתמש שבחרת)
2. לחץ **Start** או שלח `/start` / `/התחל`
3. הבוט יפתח אשף הגדרות ב-5 שלבים

## דוגמאות שימוש

### תיעוד אוכל בשפה טבעית

שלח כל הודעה חופשית עם תיאור מה אכלת:

```
אכלתי בוקר שתי ביצים עם לחם
```

```
צהריים שניצל עם אורז וסלט
```

```
בוקר קפה עם חלב, צהריים פסטה בולונז, ערב יוגורט עם פירות
```

הבוט מנתח אוטומטית:
- ✅ קלוריות
- ✅ חלבון
- ✅ פחמימות
- ✅ שומן
- ✅ זמן הארוחה (בוקר/צהריים/ערב/נשנוש)

### תכנון המשך היום

לאחר תיעוד ארוחות, שלח `/תכנון` וקבל תוכנית מותאמת אישית להמשך היום.

ניתן לבקש התאמות:
```
אני לא אוהב דגים
```
```
יש לי רק 10 דקות לבישול
```
```
אין לי גלוטן בבית
```

## נתוני SQLite

הנתונים נשמרים בקובץ `data/daybite.db`:

- **users** — פרופיל מלא לכל משתמש טלגרם
- **meal_logs** — כל הארוחות המתועדות עם תאריך
- **conversation_history** — 10 ההודעות האחרונות לכל משתמש (להקשר Claude)

יומן היום מתאפס אוטומטית בחצות.
